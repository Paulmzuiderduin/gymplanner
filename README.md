# GymPlanner

Gymplanner, buitenruimteplanner en beweegdoelenmaker in één lokale MVP voor docenten Lichamelijke Opvoeding.

## Kenmerken
- **Lessenplanner**: Plan je lessen en bewaar ze lokaal.
- **Plattegrond**: Richt je gymzaal of buitenruimte in met materialen.
- **Beweegdoelen**: Genereer beweegdoelen op basis van sportregels en doelsjablonen.
- **Privacy**: Alles draait lokaal in de browser.

## Installatie
1. Kloon de repository
2. Installeer dependencies: `npm install`
3. Start de dev server: `npm run dev`
